\section{结论与下一步}

\subsection{当前最稳的主故事}

\begin{enumerate}
\item plain \texttt{rule-only} 仍然是当前 top-10 最强的无 hint baseline，但 coverage 损失很明确，因此不适合直接当 balanced 主线。
\item \texttt{dynamic gather-fix} 应被视为当前 canonical dynamic 实现；\texttt{dynamic sid-only} 和 \texttt{ranking dynamic} 都还没有超过它。
\item clean fixed family 仍然是当前最强的 balanced 主候选；其中 corrected \texttt{fixed taskfix sid-only} 是更干净的 fixed 参考，而 old fixed 只能当 historical reference。
\item 在后续探索里，\texttt{max1} 更像 early-stop dynamic candidate；\texttt{hintce-3}、\texttt{single-hint mixed} 和 \texttt{fixed dual-task} 则分别对应 CE scaling、partial-hint 和 task-filtered 三条新的探索方向。
\end{enumerate}

\subsection{下一步最值得继续补的证据}

\begin{itemize}
\item 补 \texttt{hintce-3} 的最后一段 terminal checkpoint，确认它是不是稳态新上界，而不只是中段强峰值。
\item 继续跟 \texttt{single-hint mixed} 的尾段平台，并解释为什么它能在完整 2 epoch 维持住 fixed family 前沿。
\item 继续跟 \texttt{dynamic dual-task} 与 \texttt{fixed dual-task} 的终段 readout，确认 dual-task family 的稳定排序。
\item 如果后面要把这份稿件扩成更正式的研究报告，建议继续沿用“一个实验主题一个子文件”的结构，而不是回到按日期拼接的 note 模式。
\end{itemize}
