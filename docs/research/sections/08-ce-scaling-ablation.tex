\subsection{fixed CE family 与 dynamic CE first-look}

fixed CE family 目前仍然是这节的主体，但这次已经补进一条新的 dynamic CE first-look，因此这里统一按“4 条 fixed CE + 1 条 dynamic CE”来读：

\begin{itemize}
\item \texttt{hintce}：早期版本，这里记作 batch-mean CE。
\item \texttt{hintce-2}：和当前公式说明里 strict DAPO CE 更一致的 token-mean 版本。
\item \texttt{hintce-3}：在 \texttt{hintce-2} 的基础上，把 CE 相对倍率调到 \texttt{0.005}。
\item \texttt{hintce-4}：把 CE 相对倍率继续抬到 \texttt{0.01} 的更激进版本。
\item \texttt{dynamic hint + CE(0.005)}：在 canonical \texttt{dynamic gather-fix} 上额外挂 prompt-side CE 的新尝试；当前已经同步到 \texttt{checkpoint-999}，但还只能当 first-look。
\end{itemize}

\subsection{结论}

\begin{itemize}
\item full non-CE 仍然是当前 fixed family 的干净参考线。
\item \texttt{hintce}、\texttt{hintce-2}、\texttt{hintce-3} 和 \texttt{hintce-4} 现在已经能放到同一张图里一起读，不再只讲单条 fixed CE 线。
\item \texttt{hintce-2} 是当前更稳的 token-mean / DAPO-compatible CE 候选；\texttt{hintce-3} 则是在它基础上把倍率抬到 0.005 后得到的更激进版本。
\item 当前 `0.01` 倍率的 \texttt{hintce-4} 已经不再只是 early readout：它现在一路补到 `checkpoint-2997`，best \texttt{NDCG@10} 已经来到 `0.0953`，说明这条线不是“偏激后直接失效”，而是真正进入了可比前沿。
\item 但按当前完整可见轨迹看，\texttt{hintce-4} 和 \texttt{hintce-3} 呈现的是不同形状：\texttt{hintce-4} 更像“把 top-10 往长跑尾段继续抬高”，而 \texttt{hintce-3} 仍然保留更高的 coverage 峰值 `0.1985`。因此现在更合理的说法不是“ce4 不行”，而是“它在 top-10 上已经追平 old fixed 历史上界，但还没有在 coverage 上超过 ce3”。
\item 新同步的 \texttt{dynamic hint + CE(0.005)} 现在已经补到 \texttt{checkpoint-999}，但 best 仍停在 \texttt{checkpoint-666}：该点给出 `NDCG@10=0.0904 / HR@50=0.1910`，而到 \texttt{checkpoint-999} 回落到 `0.0899 / 0.1869`。因此当前更合理的判断仍然是：它还没有证明自己优于 canonical dynamic baseline，但已经足够像一条需要继续观察长跑后段的真实候选，而不是一次失败试验。
\end{itemize}
