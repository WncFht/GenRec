\subsection{结论}

\begin{itemize}
\item full non-CE 仍然是当前 fixed family 的干净参考线。
\item \texttt{hintce-2} 把 CE 从 early readout 拉成了一个完整轨迹上的更稳候选。
\item \texttt{hintce-3} 对应更高的 CE 相对倍率 0.005，并且已经把中段 best 点推到 \texttt{0.0945 / 0.1985}；真正还缺的是最后一段 terminal readout。
\end{itemize}
