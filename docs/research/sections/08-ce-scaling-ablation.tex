\section{CE Scaling 消融}

\subsection{四条 CE 线各自是什么}

本文按当前命名约定把 CE family 记成四条线：

\begin{itemize}
\item \texttt{hintce}：早期版本，这里记作 batch-mean CE。
\item \texttt{hintce-2}：和当前公式说明里 strict DAPO CE 更一致的 token-mean 版本。
\item \texttt{hintce-3}：在 \texttt{hintce-2} 的基础上，把 CE 相对倍率调到 \texttt{0.005}。
\item \texttt{hintce-4}：把 CE 相对倍率继续抬到 \texttt{0.01} 的更激进版本。
\end{itemize}

\subsection{结论}

\begin{itemize}
\item full non-CE 仍然是当前 fixed family 的干净参考线。
\item \texttt{hintce}、\texttt{hintce-2}、\texttt{hintce-3} 和 \texttt{hintce-4} 现在已经能放到同一张图里一起读，不再只讲单条 CE 线。
\item \texttt{hintce-2} 是当前更稳的 token-mean / DAPO-compatible CE 候选；\texttt{hintce-3} 则是在它基础上把倍率抬到 0.005 后得到的更激进版本。
\item 当前 `0.01` 倍率的 \texttt{hintce-4} 已经不再只是 early readout：它在 `checkpoint-2331` 把 \texttt{NDCG@10} 抬到 `0.0942`，说明这条线并非彻底失败。
\item 但按当前完整可见轨迹看，\texttt{hintce-4} 仍然没有超过 \texttt{hintce-3} 的 `0.0945 / 0.1985`；它更像“把 top-10 往上抬一点，但没有把 coverage 一起抬上去”的激进倍率，而不是新的最强 CE 候选。
\end{itemize}
