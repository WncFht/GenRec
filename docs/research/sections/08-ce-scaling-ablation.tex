\section{CE Scaling 消融}

\subsection{三条 CE 线各自是什么}

本文按当前命名约定把 CE family 记成三条线：

\begin{itemize}
\item \texttt{hintce}：早期版本，这里记作 batch-mean CE。
\item \texttt{hintce-2}：和当前公式说明里 strict DAPO CE 更一致的 token-mean 版本。
\item \texttt{hintce-3}：在 \texttt{hintce-2} 的基础上，把 CE 相对倍率调到 \texttt{0.005}。
\end{itemize}

\subsection{结论}

\begin{itemize}
\item full non-CE 仍然是当前 fixed family 的干净参考线。
\item \texttt{hintce}、\texttt{hintce-2} 和 \texttt{hintce-3} 现在已经能放到同一张图里一起读，不再只讲某一条 early readout。
\item \texttt{hintce-2} 是当前更稳的 token-mean / DAPO-compatible CE 候选；\texttt{hintce-3} 则是在它基础上把倍率抬到 0.005 后得到的更激进版本。
\end{itemize}
