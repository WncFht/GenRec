\subsection{结论}

\begin{itemize}
\item \texttt{dynamic dual-task} 已经进入可比较状态，但还没有超过 canonical \texttt{dynamic gather-fix}。
\item \texttt{fixed dual-task} 已经整体强于 \texttt{dynamic dual-task}，因此 dual-task family 内部也复现了 “fixed 比 dynamic 更稳” 的方向。
\item 但它离 clean fixed 主线仍然有明显差距，所以当前更准确的定位是：dual-task 是一条值得继续跟的 filtered setting，而不是已经改写主线排序。
\end{itemize}
