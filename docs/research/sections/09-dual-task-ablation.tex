\subsection{结论}

\begin{itemize}
\item 把 dual-task family 拆成 fixed / dynamic 两张图之后，fixed 侧更容易直接看出：\texttt{fixed dual-task} 仍落后于 old fixed、corrected \texttt{fixed taskfix sid-only}、corrected \texttt{fixed taskfix} 和 fixed \texttt{single-hint mixed}。
\item dynamic 侧则更容易看出：\texttt{dynamic dual-task} 已经进入可比较状态，但当前仍没有超过 canonical \texttt{dynamic gather-fix}，也没有超过 \texttt{rule-only}、\texttt{dynamic sid-only} 或 \texttt{dynamic single-hint mixed} 这些 baseline。
\item 因而当前更准确的定位仍然是：dual-task 是一条值得继续跟的 filtered setting，而不是已经改写主线排序。
\end{itemize}
