\subsection{结论}

\begin{itemize}
\item old fixed 的表面优势，不是来自更正确的 hint map，而是因为 hardest task 被系统性少给 hint。
\item 因此它更像一个带偏置的历史现象学上界，而不是应该继续复用的方法定义。
\item 这也是为什么本文后文在需要 fixed reference 时，默认优先引用 corrected \texttt{fixed taskfix sid-only}。
\end{itemize}
