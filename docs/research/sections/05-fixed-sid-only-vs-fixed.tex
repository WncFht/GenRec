\subsection{结论}

\begin{itemize}
\item \texttt{fixed taskfix sid-only} 主要把 top-10 hit 又往上推了一步，因此它是更合适的 clean fixed reference。
\item \texttt{fixed taskfix} 本身已经明显强于 plain \texttt{rule-only}，说明 task-aware 修复本身就在生效。
\item old fixed 仍然可以当历史上界参考，但不应该继续当方法定义。
\end{itemize}
