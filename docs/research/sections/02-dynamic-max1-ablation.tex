\section{Dynamic-Hint Max1 消融}

\subsection{实验问题}

这一章对应 2026-04-16 的 focused ablation。它只回答一个具体问题：

\begin{quote}
如果把 dynamic hint 的 online budget 从 \texttt{dynamic\_hint\_max\_depth=3} 收紧到 \texttt{1}，而不改 trainer 逻辑、不丢 no-hit 样本，最后的 eval trade-off 会落在什么位置。
\end{quote}

把图和表放在一起看，现象其实很一致：

\begin{enumerate}
\item \texttt{max1} 不是简单退化成 \texttt{rule-only}。它在很多 checkpoint 上保住了更高的 coverage，说明浅 budget 不是直接把 hint 机制废掉。
\item \texttt{max1} 也不是被 \texttt{gather-fix} 完全支配。它在早中段几个关键点上能给出更高的 \texttt{NDCG@10}，而且 coverage 并没有同步塌掉。
\item 但完整轨迹现在已经表明，\texttt{max1} 的优势主要停在前中段。跑到 \texttt{checkpoint-3326} 以后，它没有把优势延续到尾段，反而逐步回落。
\item 因此它更像一个值得 early-stop 的 dynamic candidate，而不是新的 long-run default。
\end{enumerate}

\subsection{本章小结}

\begin{itemize}
\item 如果后续 dynamic family 只保留一个长期默认线，当前仍然应优先保留 \texttt{gather-fix}。
\item 如果后续研究 early-stop 或 shallow-budget trade-off，\texttt{max1} 仍然非常值得继续跟。
\item 真正缺的不是更多 eval 点，而是 train-time diagnostics。没有日志，当前对机制的解释仍然只能停留在假说层。
\end{itemize}
