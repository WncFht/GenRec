如果只想记住一张最核心的对照表，这张就够了：

\begin{itemize}
\item \texttt{rule-only} 提供最强的无 hint top-10 baseline。
\item \texttt{dynamic gather-fix} 是“hint scaffold 有用”的干净证据，它把 coverage 拉回到了 SFT 以上。
\item clean \texttt{fixed taskfix sid-only} 则进一步把 trade-off 推到了更右上方：相对 canonical dynamic，多了 0.0009 的 \texttt{NDCG@10} 和 0.0080 的 \texttt{HR@50}。
\end{itemize}
