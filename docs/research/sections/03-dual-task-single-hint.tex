\section{Dual-Task 过滤与 Single-Hint Mixed 跟踪}

\subsection{本章研究问题}

这一章对应 2026-04-19 的 tracking note。它实际上覆盖两类新问题：

\begin{enumerate}
\item 只训练两个任务：从 mixed RL 三任务里移除 \texttt{task4}，只保留 \texttt{task1 + task5}。
\item 只 hint 一个任务：训练仍保留 mixed task，但 fixed hint 只注入 \texttt{task1}。
\end{enumerate}

这轮虽然有三个 launcher，但研究问题只有两类，因此这一章按研究问题分组，而不按 launcher 分组。

这部分当前最重要的判断是：

\begin{enumerate}
\item \texttt{single-hint mixed} 已经不只是一个有信号的新 launcher，而是开始站到 fixed family 的前沿区域。
\item 它当前 full-trace best 点 \texttt{0.0948 / 0.1958} 已经同时超过 corrected \texttt{fixed taskfix sid-only} 的 \texttt{0.0945 / 0.1935}。
\item 它对 old fixed 仍然略差一点 top-10，但 coverage 已经更高，因此后续真正值得解释的是：为什么它能在完整尾段维持住这个高位平台。
\end{enumerate}

现在这条线已经可以明确写成：

\begin{itemize}
\item fixed dual-task 不再只是 early look，而是一条完整可比轨迹。
\item 相比 dynamic dual-task，它不再只是交换 top-10 与 coverage，而是整体略占优。
\item 但它离 clean fixed 主线仍然有明显差距，所以更准确的定位是“dual-task 版本里 fixed 比 dynamic 更强”，而不是“dual-task 已经进入 fixed 主线最前排”。
\end{itemize}

\subsection{本章小结}

\begin{enumerate}
\item \texttt{single-hint mixed} 是这一轮最值得继续追的主候选之一，因为它已经在完整尾段维持住了 fixed family 前沿水平。
\item fixed dual-task 已经可以当作“比 dynamic dual-task 更强”的 clean 结论，但它还没有打入当前 fixed 主线最前排。
\item \texttt{hintce-3} 已经从 early readout 升级成中段强峰值候选，后续只差最后一段 terminal readout 来判断它是不是稳态新上界。
\end{enumerate}
