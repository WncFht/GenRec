\subsection{结论}

\begin{itemize}
\item 这条线不是“只训练 task1”，而是 mixed 三任务仍然训练，只是 fixed hint 只注入到 \texttt{task1\_sid\_sft}。
\item 它当前 best 点已经同时超过 corrected \texttt{fixed taskfix sid-only} 的 \texttt{NDCG@10} 和 \texttt{HR@50}。
\item 因此后续更值得解释的问题不是“它有没有信号”，而是它为什么能在完整尾段维持住 fixed family 前沿水平。
\end{itemize}
